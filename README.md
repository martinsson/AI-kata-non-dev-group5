# AgroAssistant

Assistant web pour diagnostiquer les maladies des plantes et obtenir des conseils agronomiques.

Edit files in `site/`, push to `main`, GitHub Actions deploys to Pages.

One-time per fork/clone: **Settings → Pages → Source: GitHub Actions**.

## Fonctionnalités

- **Diagnostic** : sélectionnez votre plante et vos symptômes → obtenez des hypothèses de maladies avec traitements recommandés
- **Mon jardin** : suivez l'état de santé de vos plantes
- **Historique** : retrouvez vos diagnostics passés
- **Guide des maladies** : base de connaissances consultable (mildiou, oïdium, botrytis, pucerons…)
